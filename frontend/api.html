How to Use the Widget Builder

1
Get Your API Key

Use the Your API Key section above to connect your wallet and generate a free API key. This key authenticates your widget and is automatically embedded in the code you copy.

2
Enter a Token Address

Paste any Solana token mint address into the field above. The builder will fetch live data and display it in the preview panel on the right.

3
Choose Your Data

Toggle the data modules you want shown: price, market cap, holder analytics, sentiment, community links, and more. The preview updates instantly.

4
Customize Appearance

Pick a dark or light theme, adjust colors, border radius, and width to match your site's design. Set an auto-refresh interval to keep data current.

5
Copy & Embed

Click Copy Code to grab the embed snippet. Paste it into any HTML page — your API key is already included. The widget loads asynchronously and won't block your page.

API Key in the embed code: If you have an active API key, it is automatically inserted into the generated snippet. If you see YOUR_API_KEY, generate a key first using the section above.
Rate limits apply: Each widget refresh counts as API requests against your key's 100 requests/minute limit. Choose a reasonable refresh interval for your traffic level.
Responsive: The widget respects max-width: 100%, so it adapts to smaller containers. Set the width to your preferred maximum size.

API Endpoints

All API requests require your API key in the X-API-Key header.

GET /api/v1/community/:mint

Get community content (banner, social links) for a token

Example Request
curl -H "X-API-Key: YOUR_KEY" \
  https://opendex-api-dy30.onrender.com/api/v1/community/So11111111111111111111111111111111111111112
Response
{
  "success": true,
  "data": {
    "token": "So11111111111111111111111111111111111111112",
    "banner": {
      "url": "https://i.imgur.com/example.png",
      "score": 15
    },
    "links": {
      "twitter": { "url": "https://twitter.com/solana", "score": 12 },
      "telegram": null,
      "discord": { "url": "https://discord.gg/solana", "score": 8 },
      "tiktok": null,
      "website": { "url": "https://solana.com", "score": 10 }
    }
  }
}
GET /api/v1/community/:mint/all

Get all approved submissions for a token (not just top-voted)

Query Parameters
type    (optional) - Filter by type: banner, twitter, telegram, discord, tiktok, website
GET /api/v1/community/batch

Get community content for multiple tokens at once (max 20)

Query Parameters
mints   (required) - Comma-separated token addresses (max 20)
Example Request
curl -H "X-API-Key: YOUR_KEY" \
  "https://opendex-api-dy30.onrender.com/api/v1/community/batch?mints=mint1,mint2,mint3"
GET /api/v1/keys/info

Get information about your API key (usage stats, creation date)

Sentiment Endpoints

GET /api/sentiment/:mint

Get bullish/bearish sentiment tally for a token

Query Parameters
wallet   (optional) - Include your vote in the response
Response
{
  "tally": { "bullish": 45, "bearish": 15, "score": 30 },
  "userVote": { "sentiment": "bullish" }
}
POST /api/sentiment/:mint

Cast a bullish or bearish vote (no holder verification required, wallet connection only)

Request Body
{ "voterWallet": "...", "sentiment": "bullish | bearish" }

OG Finder

GET /api/ogfinder/search

Search PumpFun tokens by name or symbol, sorted by creation date (oldest first)

Query Parameters
q   string   Search query (name or symbol)

Rate Limits

API requests are rate limited to prevent abuse:

  • 100 requests/minute - Standard endpoints
  • 10 requests/minute - Key registration

Batch endpoints (e.g., /community/batch) count as 1 request regardless of how many tokens are queried.

Error Codes

401 Missing or invalid API key
403 API key has been revoked
429 Rate limit exceeded
503 Service temporarily unavailable
